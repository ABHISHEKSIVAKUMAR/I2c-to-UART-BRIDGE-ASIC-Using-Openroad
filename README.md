# I2c-to-UART-BRIDGE-ASIC-Using-Openroad
This project demonstrates a complete RTL-to-GDSII implementation of an I2C-UART bridge ASIC using the open-source OpenLane/OpenROAD physical design flow and the SKY130 PDK.
## 📄 Full Documentation

A complete 19-page technical report containing:

- RTL simulation methodology
- OpenLane/OpenROAD flow
- Floorplanning and placement results
- CTS and routing reports
- Static Timing Analysis
- Power analysis
- DRC/LVS verification
- GDSII generation and visualization
